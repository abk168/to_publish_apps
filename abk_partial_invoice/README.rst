# Custom Fixed Amount Partial Invoice for Odoo 18
===============================================

This module enables **flexible partial invoicing** in Odoo 18 by allowing users to
create invoices using a **fixed amount** or a **percentage** of the Sale Order total.
It also supports **invoicing from Delivery Orders (DN)** with a **custom invoice report layout**.

---

## 🔑 Key Features

- ✅ Create partial invoices using **fixed amount** or **percentage**
- ✅ Generate invoices directly from **Delivery Orders (DN)**
- ✅ Supports **multiple delivery orders** per Sale Order
- ✅ Clear visibility of invoiced and remaining amounts
- ✅ Prevents over-invoicing automatically
- ✅ Fully integrated with Sale Orders, Deliveries, and Invoices
- ✅ Multi-company compatible
- ✅ Access control via standard Odoo security groups
- ✅ Compatible with **Odoo 18 Enterprise

---

## 🛠️ How It Works

### 🔹 1. Sale Order Confirmation

1. Go to **Sales > Orders**
2. Create and confirm a **Sale Order**
3. The order becomes ready for partial invoicing

---

### 💰 Partial Invoice (Fixed Amount / Percentage)

1. From the **Sale Order**, click **Create Invoice**
2. Choose **Down Payment**
3. Select:
   - **Fixed Amount**, or
   - **Percentage of Sale Order**
4. Enter the value and confirm
5. A **draft invoice** is created with the specified amount

---

### 🧾 Draft Invoice Review

1. Open the generated **Draft Invoice**
2. Review:
   - Invoice lines
   - Partial amount
   - Taxes and totals
3. Validate the invoice when ready

---

### 📦 Create Invoice from Delivery Order (DN)

1. Open the related **Delivery Order**
2. Validate the delivery
3. Click **Create Invoice**
4. The system creates a **draft invoice** based on delivered quantities
5. Review and validate the invoice
